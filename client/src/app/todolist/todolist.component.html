    • One should be able to use the YouTube video id to embed the video in the slider. The video id is in the link for the video. For this video link "https://youtu.be/WbRlW58J9SE?si=3nIyxxSGBA_DFQ2v" the id would be: "WbRlW58J9SE".

      This is how youtube videos can be embedded:
      iframe width="560" height="315" src="https://www.youtube.com/embed/WbRlW58J9SE?si=3nIyxxSGBA_DFQ2v" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" referrerpolicy="strict-origin-when-cross-origin" allowfullscreen /iframe
